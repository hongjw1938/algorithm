### 그래프
- 그래프를 생성 및 초기화시 간선의 방향성 여부를 선택하고 가중치의 부여 여부 선택 가능. 이후에는 정점과 간선을 삽입 및 삭제 가능
- 정점(vertex) : 연결의 대상이 되는 개체 또는 위치
- 간선(edge) : 정점 사이의 연결
- 표현 방법
	- 인접 행렬 기반 그래프 : 정방 행렬을 이용
		- 정점에 개수에 따른 2차원 정방행렬을 생성한다. 두 정점의 연결 여부에 따라 연결되어있으면 1, 아니면 0으로 표현한다.
		- 무방향 그래프의 경우 간선의 방향성이 없기 때문에 행렬이 대각선을 기준으로 대칭을 이루게 된다.
	- 인접 리스트 기반 그래프 : 연결 리스트 이용
		- 각각의 정점에 연결된 간선의 정보를 각각의 연결 리스트에 담는다.
- 최소 신장 트리(MST)
	- 신장트리는 그래프의 모든 정점을 연결하는 트리를 의미한다. 즉, 그래프의 하위 개념
	- 최소 신장 트리 알고리즘은 "최소한의 비용으로 모든 정점을 연결하는 간선을 찾아 각각을 연결하는" 알고리즘을 의미한다.
	- 이를 해내기 위한 알고리즘은 프림 알고리즘과 크루스칼 알고리즘이 있다.
	- 프림알고리즘(PrimMST참조)
		- 로버트 프림이 고안한 그래프에서 최소 신장 트리를 만들어내는 알고리즘이다.
		- 방법
			1. 그래프와 최소 신장 트리를 준비한다. 최소 신장 트리는 노드가 하나도 없다.
			2. 그래프에서 임의의 정점을 시작 정점으로 선택해 최소 신장 트리의 루트 노드로 삽입한다.
			3. 최소 신정 트리에 삽입되어 있는 정점들과 이 정점들의 모든 인접 정점 사이에 있는 간선의 가중치를 조사함. 간선 중에 가장 가중치가 작은 것을 골라 이 간선에 연결되어 있는 인접 정점을 최소 신장 트리에 삽입한다.(기존 노드와 사이클 형성하면 안됨)
			4. 3을 반복하다가 모든 정점을 연결하게 되면 알고리즘 종료
